rbcb 0.1.3 - Como usar?

Acabei de submeter ao CRAN a versão 0.1.3 do rbcb https://github.com/wilsonfreitas/rbcb. Está versão está com algumas funcionalidades que eu já queria a algum tempo, como baixar múltiplas séries e nomerar as séries no argumento code eliminando assim o argumento name da função get_series. Outra coisa interessante é que agora eu consigo baixar um metadado da série que me permiter saber a frequência da série (diária, mensal ou anual). Assim é possível gerar os objetos ts na frequência correta sem necessidade de parametrização pelo usuário.
